Write a program in any language to find the maximum and minimum element in an array.


C++ code for given problem statement:-

            

            // C++ program to find the maximum and minimum element in an array.
            #include
              using namespace std;
              int main ()
              {
                  int arr[10], n, i, max, min;
                  cout << "Enter the size of the array : ";
                  cin >> n;
                  cout << "Enter the elements of the array : ";
                  for (i = 0; i < n; i++)
                      cin >> arr[i];
                  max = arr[0];
                  for (i = 0; i < n; i++)
                  {
                      if (max < arr[i])
                          max = arr[i];
                  }
                  min = arr[0];
                  for (i = 0; i < n; i++)
                  {
                      if (min > arr[i])
                          min = arr[i];
                  }
                 
                  cout << "Minimum of array is : " << min;
                  cout << "Maximum of array is : " << max;
                  return 0;
              }

SAMPLE INPUT:

Enter the size of the array : 5
Enter the elements of the array :14 56 89 65 23


SAMPLE OUTPUT:

Minimum of array is : 14
Maximum of array is : 89


VARIABLES USED:

arr, n, i, max, min

COMPLEXITY:

Time complexity of the above program is: O(n)
